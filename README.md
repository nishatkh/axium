# Axium — Premium Engineering Dashboard & Component Library

![Axium Banner](https://img.shields.io/badge/AESTHETIC-PURE_BLACK-000000?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/STACK-REACT_19_%7C_VITE_%7C_TAILWIND_V4-blue?style=for-the-badge)

Axium is a high-fidelity, performance-optimized production platform designed for engineers who value precision, aesthetics, and speed. It serves as both a high-end component library and a showcase for state-of-the-art web design.

---

## 🚀 Key Features

### 1. Cinematic Engineering Hero
*   **Next-Gen Hardware Mockup**: Features an iPhone 17 Pro Max chassis with a custom-rendered engineering dashboard.
*   **Liquid Glass Refraction**: Real-time refraction effects and film grain overlays for a tactile, cinematic feel.
*   **GSAP Powered Orchestration**: Complex, coordinated entrance animations for taglines and UI elements.
*   **Pure Black Aesthetic**: Optimized for OLED displays with true #000000 backgrounds.

### 2. Website Design Gallery (Live Previews)
*   **Interactive Showcases**: Browse a curated list of full-page design systems including:
    *   **Minimalist Monochrome**: Stark, editorial, high-contrast serif design.
    *   **Bauhaus UI**: Functionalist, geometric, and primary color-driven.
    *   **Luxury Editorial**: Sophisticated, slow-motion, alabaster-toned fashion aesthetic.
    *   **Terminal CLI**: Developer-focused, monospace, hacker-themed interface.
    *   **Swiss Style**: Mathematics-driven grid system with neutral typography.
*   **Direct Prompt Access**: Every design includes the exact AI system prompt used to generate the layout.

### 3. AI Image Prompt Engine
*   **High-Density Grid**: Optimized image gallery for discovering and copying high-end AI prompts.
*   **Real-time Search**: Instant filtering of prompts and assets.
*   **One-Click Copy**: Streamlined workflow to "Get Prompt" and move directly into production.

### 4. Advanced Component Library
*   **Interactive Keyboard Demo**: A fully functional mechanical keyboard simulation with real-time sound effects (Mechanical Blue switches).
*   **Dynamic Text Rotation**: Smooth typographic transitions for high-impact taglines.
*   **Animated Testimonial Grid**: Performance-optimized grid with hover-triggered lift effects and grayscale-to-color transitions.
*   **Dynamic Island ToC**: A modern, responsive Table of Contents inspired by the mobile Dynamic Island interaction.
*   **Shine Buttons & Glow Effects**: Premium interactive elements with light-tracking borders and internal glows.

### 5. Technical Directories
*   **MCP Directory**: A comprehensive guide to Model Context Protocol integrations.
*   **Color Systems**: Visualization of professional design tokens and accessible color palettes.
*   **Icon Library**: Deep integration with Lucide React for consistent technical iconography.
*   **Three.js & 3D Elements**: Integration of low-latency 3D elements for spatial UI.

---

## 🛠 Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **React 19** | Core UI Library with concurrent rendering |
| **Vite 6** | Lightning-fast build tool and development server |
| **Tailwind CSS v4** | Next-generation utility-first styling with native CSS variables |
| **GSAP** | Advanced timeline-based animations and scroll orchestration |
| **Framer Motion** | Declarative layout transitions and micro-interactions |
| **TypeScript** | Type-safe development for enterprise-grade stability |
| **Lucide React** | Consistent, high-fidelity iconography |

---

## 🎨 Design Philosophy

*   **Precision over Decoration**: Every pixel, line, and border weight is calculated for clarity.
*   **Motion with Purpose**: Animations are either snappy (100ms) for utility or cinematic (1500ms+) for storytelling.
*   **Accessibility**: High-contrast ratios (AAA compliance) and optimized typography for long-form reading.
*   **Performance**: Zero-latency transitions, image optimization, and modular code splitting for sub-second load times.

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
